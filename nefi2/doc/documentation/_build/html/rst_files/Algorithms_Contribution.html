Algorithms Contribution¶

This document is a guide to add your own new algorithm in Nefi 2.0. The guide provides you all the step to be done for allow the users of Nefi 2.0 to use the new algorithm.

Introduction¶

First of all you have to create a .py file in the folder module.algorithms. This file will contain the algorithm that you want to add. As convention of Nefi 2.0, the name of the file will be the name of the algorithm, for example if the new algorithm is Blur the file’s name will be “blur.py”. The main structure of the file will be a main class AlgBody and its main methods AlgBody.__init__() and AlgBody.process().

Imports¶

You have to import from module.algorithms._alg the class model.algorithms._alg.Algorithm. Also you have to import, if the algorithm needs it, the UI widgets that allow the user to select the input values of the algorithm. The UI widgets that Nefi 2.0 provides are:

  • model.algorithms._alg.IntegerSlider.
  • model.algorithms._alg.FloatSlider.
  • model.algorithms._alg.CheckBox.
  • model.algorithms._alg.DropDown.

AlgBody Class¶

The AlgBody class will contain all the methods that you need to implement your algorithm and the methods __init__() and process().

The __init__() method is necessary to set the name of the algorithm, to link the algorithm at its category and to set the UI widgets.

The name of the algorithm is defined by the attribute self.name.

The parent category of the algorithm is defined by the attribute self.parent.

To add the UI widgets create an UI object and then append it to the list self.integer_sliders, self.float_sliders, self.checkboxes or self.drop_downs.

The process() is the main method that contains the logic of the algorithm.

The results of the algorithm must be saved in the object self.result['img'], if it is an image, and/or in self.result['graph'], if it is a graph.
