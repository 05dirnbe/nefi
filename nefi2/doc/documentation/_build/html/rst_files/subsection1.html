_category¶

class model.categories._category.Category(name, icon=None)¶

This class represents image processing method that contains its respective algorithms. Its main function is controlling an algorithm, collecting and transmitting the output to the pipeline. It serves as an intermediate layer between the algorithms and the pipeline.
